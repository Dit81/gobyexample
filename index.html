Go в примерах

Go - это язык программирования с открытым исходным кодом, предназначенный для создания простого, быстрого, и надежного программного обеспечения.

Go в примерах - это практическое введение в Go использущее аннотированные примеры программ. Посмотрите первый пример или весь список примеров ниже.

  • Hello World
  • Типы данных (Values)
  • Переменные (Variables)
  • Константы (Constants)
  • For
  • If/Else
  • Switch
  • Массивы (Arrays)
  • Срезы (Slices)
  • Направления каналов (Channel Directions)
  • Оператор выбора для каналов (Select)
  • Тайм-ауты (Timeouts)
  • Неблокирующие операции с каналами (Non-Blocking Channel Operations)
  • Закрытие каналов (Closing Channels)
  • Перебор значений каналов (Range over Channels)
  • Таймеры (Timers)
  • Счётчики тиков (Tickers)
  • Набор обработчиков (Worker Pools)
  • Ограничение скорости (Rate Limiting)
  • Атомарные счётчики (Atomic Counters)
  • Мьютексы (Mutexes)
  • Отложенный вызов
  • Функции коллекции
  • Функции для работы со строками
  • Форматирование строк
  • Линейные фильтры (Line Filters)
  • Аргументы командной строки (Command-Line Arguments)
  • Флаги командной строки (Command-Line Flags)
  • Переменные окружения (Environment Variables)
  • Порождение процессов (Spawning Processes)
  • Выполнение процессов (Exec'ing Processes)
  • Сигналы (Signals)
  • Выход (Exit)
